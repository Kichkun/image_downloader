# Image downloader by query

## Installation.

```bash
git clone https://github.com/Kichkun/image_downloader.git
cd race_model
pip install .
```

**Usage**

'-s' - 'Phrase for search'

'-c' - 'Path to chromedriver'

'-d' - 'Save images with faces only'

'-o' - 'Location of output results'

**Example**
```bash
python parser.py -s 'your request'
```
